Welcome to Contact List!

Comandos disponíveis

  • GET /list - Lista todos os registros
  • GET /contact?id=x - Lista o registro pelo id
  • POST /?name=x&description=y - Inclui registro
  • PUT /?id=x&name=y&description=z - Atualiza registro
  • DELETE /contact?id=x - Deleta o registro pelo id
