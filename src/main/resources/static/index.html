Welcome to Contact List!

Comandos disponíveis

  • GET /users - Lista todos os usuarios

  • GET /contacts - Lista todos os contatos

  • GET /users/{uid}/contacts - Lista todos os contatos de um usuario

  • GET /users/{uid}/contacts/{id} - Lista um contato pelo id

  • POST /users/
    body: {"uid": "xxxx"} - Inclui usuario

  • POST /users/{uid}/contact
    body: {"name": "x", "phone": "y", "email": "z"} - Inclui registro

  • PUT /users/{uid}/contacts/{id}
    body: {"name": "x", "phone": "y", "email": "z"} - Atualiza registro

  • DELETE /users/{uid}/contacts/{id} - Deleta o registro pelo id
